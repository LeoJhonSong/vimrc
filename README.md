## Installation

### Linux

The first line download this repository to your **~/.vim/**,
the second line links **~/.vim/vimrc** with **~/.vimrc**

```shell
git clone git@github.com:LeoJhonSong/vimrc.git ~/.vim
ln -s ~/.vim/vimrc ~/.vimrc
```
